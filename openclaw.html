⚙️ Setup Guide

Get all 4 actors running in 4 steps

1
Install OpenClaw
Run the OpenClaw onboarding wizard. It guides you through setting up the gateway, connecting channels, and adding skills.
# Install and run interactively npx openclaw onboard # Or install globally npm install -g openclaw openclaw onboard
ℹ️ Requires Node.js 18+. Works on macOS, Linux, and Windows (WSL2).
2
Add AgriVN Skills to OpenClaw Config
Register the AgriVN skills file with your OpenClaw gateway. This gives each actor access to AgriVN's tools.
# Add to ~/.openclaw/config.yaml skills: - path: ./openclaw-skills.js name: agrivn actors: farmer: channels: [zalo, telegram] language: vi skill: agrivn buyer: channels: [telegram, whatsapp] language: en skill: agrivn logistics: channels: [telegram, slack] skill: agrivn admin: channels: [slack, discord] skill: agrivn
3
Connect Channels
Add the channels your farmers and buyers already use. Zalo is critical for Vietnamese farmers — OpenClaw supports both Zalo Business and Zalo Personal.
# Zalo (for Vietnamese farmers) openclaw channel add zalo --actor farmer # Telegram (for global buyers) openclaw channel add telegram --actor buyer # WhatsApp (for global buyers) openclaw channel add whatsapp --actor buyer # Slack (for logistics + admin team) openclaw channel add slack --actor logistics openclaw channel add slack --actor admin
ℹ️ For Zalo: you need a Zalo OA (Official Account) or use Zalo Personal with the OpenClaw Zalo bridge.
4
Connect to AgriVN + ERPNext
Set the environment variables so OpenClaw agents can call your AgriVN API and ERPNext instance for live data.
# Add to .env (or export in your shell) AGRIVN_API_BASE=https://agrivn-platform.vercel.app ERPNEXT_URL=https://your-erp.erpnext.com ERPNEXT_API_KEY=your_api_key_here ERPNEXT_API_SECRET=your_api_secret_here # Start the gateway openclaw start # Verify actors are registered openclaw status
